│                                                                              │
│ Context                                                                      │
│                                                                              │
│ Promptdis has SDKs in Python (sdk-py/), TypeScript (sdk-ts/), and Go         │
│ (sdk-go/). The TypeScript SDK is ESM-only and requires TypeScript tooling. A │
│  pure JavaScript SDK in sdk-js/ will serve developers who want a zero-build, │
│  no-TypeScript dependency client that works in both ESM and CommonJS         │
│ environments. The JS SDK is a direct port of the TS SDK with JSDoc           │
│ annotations instead of TypeScript types.                                     │
│                                                                              │
